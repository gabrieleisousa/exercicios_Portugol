O objetivo desse repositório é armazenar todos os exercícios realizados em Portugol para treinar lógica de programação. 
Comandos utilizados para para clonar o repositório: com a pasta aberta no gitbash e com o link do repositório, utilizo o comando git clone seguido do link do repositório.
Para versionar o repositório é utilizado os seguintes comandos: git add seguido do nome do arquivo ou da pasta que desejo enviar os arquivo, git commit -m seguido do comentário sobre o que estou enviando para o github entre aspas e utilizo o git push -u origin main para o primeiro commit, para os próximos basta digitar git push.
